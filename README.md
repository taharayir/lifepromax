# ⚔️ QuestLife

**زندگی واقعی‌ات = دنیای یک بازی RPG**

QuestLife یک Habit Tracker نیست — یک بازی نقش‌آفرینی کامله که توش زندگی واقعی‌ات رو به یک سفر حماسی تبدیل می‌کنی.

---

## ✨ امکانات

| سیستم | توضیح |
|---|---|
| 🧙 **شخصیت RPG** | Level up کن، XP بگیر، HP داشته باش |
| ⚔️ **سیستم مأموریت** | مأموریت روزانه + سفارشی‌سازی |
| 👾 **Boss System** | هر عادت = ضربه به باس! |
| 🌳 **Skill Tree** | درخت مهارت برای STR / INT / WIS / ENG |
| 🗺️ **World Map** | نقشه جهان با مناطق قابل باز شدن |
| 🛒 **Shop** | خرید آیتم با Gold |
| 🎯 **Daily Challenge** | چالش روزانه با جایزه |
| 🔥 **Streak System** | ضریب XP بیشتر با streak بالاتر |
| 🏆 **Achievements** | ۱۵ دستاورد مختلف |
| ✨ **Particles** | پس‌زمینه متحرک |

---

## 🚀 نصب روی آیفون

1. فایل `index.html` رو در **Safari** باز کن
2. دکمه **Share** رو بزن
3. **Add to Home Screen** رو انتخاب کن
4. بزن **Add** — حالا مثل یه اپ نصبه! ✅
